# Choice - Random Card Drawing App

An interactive random card drawing application built with React + TypeScript + Vite.

## Getting Started

### Prerequisites
- Node.js >= 16
- npm or yarn

### Installation
```bash
npm install
```

### Development
```bash
npm run dev
```
Visit http://localhost:5173

### Build for Production
```bash
npm run build
```

### Preview Production Build
```bash
npm run preview
```
